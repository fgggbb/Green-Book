# Green Book

**Green Book** 是基於 [c001apk-compose](https://github.com/bggRGjQaUbCoE/c001apk-compose) 修改的酷安（Coolapk）第三方客戶端。

> 原始專案已被作者刪除，此為備份修改版。

## 修改內容

- 更新版本號至酷安官方最新版（16.2.1）
- 移除設備偽裝，改為讀取真實設備參數
- Home 介面標籤漢化（臺灣用語）
- GitHub 連結更新至此倉庫
- 應用名稱改為 Green Book

## 原始專案

原始專案 c001apk-compose 是一個使用 Jetpack Compose 開發的酷安第三方客戶端，採用 Kotlin 語言。
